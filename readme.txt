Para rodar o projeto além das dependencias contidas no package.json

É necessário ter o Docker instalado e rodando (cmd: docker compose up -d)

E a utilização do https://hoppscotch.io/ para os procedimentos de requests (post, get, etc)

Nesse projeto  foi utilizada as seguintes tecnologias:

- Node
- Typescript
- Fastify
- Docker
- Prisma
- Zod
- Websockets
